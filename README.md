# 💣 Minesweeper

ブラウザで動作するマインスイーパです。インストール不要、単一ファイルで完結します。

## 起動方法

`index.html` をブラウザで開くだけです。

## 操作方法

| 操作 | 内容 |
|------|------|
| 左クリック | マスを開く |
| 右クリック | 旗を立てる / 解除する |

- 最初のクリックは必ず安全（その周囲に地雷は配置されません）
- 数字は周囲8マスにある地雷の数を示します

## 勝利・敗北条件

- **勝利**: 地雷以外の全マスを開く
- **敗北**: 地雷を踏む

## 難易度

| 難易度 | グリッド | 地雷数 |
|--------|----------|--------|
| 初級 | 9 × 9 | 10 |
| 中級 | 16 × 16 | 40 |
| 上級 | 30 × 16 | 99 |

## ファイル構成

```
minesweeper/
└── index.html   # ゲーム本体（HTML / CSS / JS をすべて含む）
```
